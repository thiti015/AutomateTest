# วิธีเริ่มต้นการทดสอบด้วย Playwright

## 1. ติดตั้ง Dependencies

### ตรวจสอบว่าติดตั้งแล้วหรือยัง
รันคำสั่งนี้เพื่อตรวจสอบว่าได้ติดตั้ง dependencies แล้วหรือยัง:

```
npm ls
```
ถ้าไม่มี error แสดงว่า dependencies ถูกติดตั้งแล้ว

หากยังไม่ได้ติดตั้ง ให้รันคำสั่งนี้ในโฟลเดอร์โปรเจกต์:

```
npm install
```

## 2. ติดตั้ง Playwright (ถ้ายังไม่ได้ติดตั้ง)

### ตรวจสอบว่า Playwright ติดตั้งแล้วหรือยัง
รันคำสั่งนี้:

```
npx playwright --version
```
ถ้าแสดงเลขเวอร์ชัน แปลว่าติดตั้งแล้ว

หากยังไม่ได้ติดตั้ง ให้รันคำสั่งนี้:

```
npm init playwright@latest
```

## 3. ตรวจสอบไฟล์ทดสอบ

ไฟล์ทดสอบจะอยู่ในโฟลเดอร์ `tests/` เช่น
- tests/login.spec.ts

## 4. รันเทสทั้งหมด

```
npx playwright test --ui
```

## 5. รันเทสเฉพาะไฟล์

```
npx playwright test tests/login.spec.ts
```

## 6. ดูผลลัพธ์การทดสอบแบบ UI

หลังจากรันเทส สามารถดูรายงานแบบ UI ได้ด้วยคำสั่ง:

```
npx playwright show-report
```

## 7. ปรับแต่งไฟล์ config

สามารถตั้งค่าต่าง ๆ ได้ในไฟล์ `playwright.config.ts`

---

**หมายเหตุ:**
- หากพบปัญหา ให้ตรวจสอบว่าได้ติดตั้ง dependencies และตั้งค่าไฟล์ config ถูกต้องแล้ว
